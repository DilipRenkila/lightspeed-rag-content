Telco RAN DU {product-version} reference design overview

The Telco RAN distributed unit (DU) {product-version} reference design configures an "Red Hat OpenShift Container Platform" {product-version} cluster running on commodity hardware to host telco RAN DU workloads. It captures the recommended, tested, and supported configurations to get reliable and repeatable performance for a cluster running the telco RAN DU profile.

Deployment architecture overview
You deploy the telco RAN DU {product-version} reference configuration to managed clusters from a centrally managed RHACM hub cluster. The reference design specification (RDS) includes configuration of the managed clusters and the hub cluster components.

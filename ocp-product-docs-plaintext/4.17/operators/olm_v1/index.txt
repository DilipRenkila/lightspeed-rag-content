Operator Lifecycle Manager (OLM) has been included with Red Hat OpenShift Container Platform 4 since its initial release. Red Hat OpenShift Container Platform 4.17 includes components for a next-generation iteration of OLM as a Technology Preview feature, known during this phase as OLM v1. This updated framework evolves many of the concepts that have been part of previous versions of OLM and adds new capabilities.

Starting in Red Hat OpenShift Container Platform 4.17, documentation for OLM v1 has been moved to the following new guide:
